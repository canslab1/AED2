All notable changes to this project will be documented in this file.

## v1.0.0 (2026-03-09)

Initial public release.

### Added
- Core C++ genetic algorithm for optimal AED placement
- Spatial weighting with distance decay model (8 grid layers)
- Adaptive stirring operator for escaping local optima
- Jupyter notebooks for figure generation and statistical analysis
- Input data: 1,625 OHCA cases, 677 7-Eleven locations, 44 fire stations
- Publication-ready visualization outputs
- MIT License and comprehensive README documentation
